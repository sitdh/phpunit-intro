# Code template

## Evaluation Grade block

```php

$gradeLetter = 'F';

if ($netScore > 91):
  $gradeLetter = 'A+';
  
elseif ($netScore > 81):
  $gradeLetter = 'A';
  
elseif ($netScore > 71):
  $gradeLetter = 'B';
  
elseif ($netScore > 61):
  $gradeLetter = 'C';
  
elseif ($netScore > 51):
  $gradeLetter = 'D';
  
endif;
```
